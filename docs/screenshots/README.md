## Képernyőkép készítési tippek

1. **Windows:** `Win + Shift + S` (Snipping Tool)
2. **Böngésző DevTools:** `F12` majd Network fül
3. **Teljes oldal:** Böngésző screenshot funkció vagy extension
4. **PNG formátum:** Jobb minőség a dokumentációhoz
